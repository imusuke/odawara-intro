# 諸白小路｜小田原城下の歴史的町名

小田原城下南西部の武家地にあった諸白小路について、史料・地名由来・現在の位置などをまとめたWebページです。

## デプロイ方法（Vercel）

### 方法1: Vercel CLI（推奨）

1. [Vercel](https://vercel.com) にログイン
2. ターミナルで以下を実行：

```bash
npx vercel login
npx vercel --prod
```

### 方法2: Vercel Web UI

1. [vercel.com](https://vercel.com) にアクセスし、GitHub/GitLab/Bitbucket でログイン
2. 「Add New Project」をクリック
3. このリポジトリをインポート
4. ビルド設定はそのまま（静的サイトとして自動検出）
5. 「Deploy」をクリック

### 方法3: GitHub連携

このリポジトリをGitHubにプッシュした後、Vercelダッシュボードからリポジトリを接続すると、コミットのたびに自動デプロイされます。
